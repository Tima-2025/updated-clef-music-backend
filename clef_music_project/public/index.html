üéØ API Endpoints

The system provides these API endpoints:

  • POST /api/visitors/track - Track a visitor
  • GET /api/visitors/stats - Get visitor statistics (admin only)
  • GET /api/visitors/notifications/:userId - Get user notifications
  • PATCH /api/visitors/notifications/:notificationId/read - Mark notification as read
